
# Clicker Simulator

A little game that you can click and earn click points, which you can use to do upgrades to your click power or to autoclick.


## What you can do

- When clicking to the click button you earn click points, which can be multiplied with click upgrades.

## Stack utilized

**Front-end:** Html, css and JavaScript.
